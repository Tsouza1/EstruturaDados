<---- Jogo de Cartas ---->

Jogo de Cartas implementado em node javascript, utilizando lista duplamente encadeada circular.

Inicialmente deve-se ter o Node.js instalado em sua maquina.

Para iniciar o jogo, apenas abra o prompt de comando, entre com o caminho para aonde o arquivo do jogo se localiza, digite no prompt "node linkedCards.js", divirta-se !!
